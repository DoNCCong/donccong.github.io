### Vòng lặp While là gì?
- Vòng lặp while lặp qua một khối mã miễn là điều kiện đã chỉ định là đúng(true)
```shell
while (condition) {
  // code block to be executed
}
```
- Ví dụ:
```shell
int i = 0;
while (i < 5) {
  cout << i << "\n";
  i++;
}
```
