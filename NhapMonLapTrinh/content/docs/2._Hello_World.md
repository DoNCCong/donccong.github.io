### Giới thiệu về cấu trúc file trong C++.
- Chương trình C++ sẽ có dạng như sau:
```shell
#include<iostream>
using namespace std;

int main(){
	// do something
	return 0;
}
```

- Khai báo thư viện trong C++
```shell
#include<iostream>
```

- Sử dụng namespace trong C++
```shell
using namespace std;
```

- Phần thân chương trình:
```shell
int main(){
	// do_something
	return 0;
}
```

- // do_something đây là nơi ta bắt đầu viết câu lệnh
