### String
Chuỗi được sử dụng để lưu trữ văn bản. Biến string chứa tập hợp các ký tự được bao quanh bởi dấu ngoặc kép.

```shell
// Include the string library
#include <string>

// Create a string variable
string greeting = "Hello"; 
```
